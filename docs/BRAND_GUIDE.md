# Brand Mission

Build a premium portfolio that showcases technical skills, projects, achievements, education, and continuous learning while creating a strong first impression for universities, recruiters, and employers.

---

# Brand Personality

The portfolio should feel:

- Professional
- Modern
- Developer-focused
- Minimal
- Premium
- Trustworthy
- Elegant
- Future-oriented

Avoid:

- Cartoonish designs
- Overly colorful layouts
- Excessive animations
- Cluttered interfaces
- Unnecessary visual effects

---

# Design Inspiration

Combine the design philosophies of:

- Apple
- Vercel
- GitHub
- Linear

Do not copy any website.

Use them only as inspiration.

# Border Radius

Use rounded corners consistently.

Preferred radius:

Large

Avoid sharp corners.

---

# Spacing

Generous whitespace.

Avoid crowded layouts.

Maintain consistent spacing between sections.

---

# Shadows

Use subtle shadows only.

Avoid heavy drop shadows.

---

# Icons

Use Lucide React.

Maintain a consistent icon style throughout the website.

---

# Buttons

Primary Button

Filled

Purple

Rounded

Hover animation

Secondary Button

Outlined

Transparent

Rounded

---

# Cards

Cards should have:

- Rounded corners
- Thin border
- Soft hover effect
- Consistent padding

Avoid heavy shadows.

---

# Animations

Animations should feel smooth and premium.

Preferred:

- Fade
- Slide
- Scale
- Blur transitions

Avoid:

- Bounce
- Flash
- Spin
- Excessive motion

Animations should enhance usability, not distract.
